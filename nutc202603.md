## 建立目標

建立 31 個 SQL Server Login，名稱為 `nutc00` ~ `nutc30`，預設密碼統一設定。

## 執行 SQL

```sql
DECLARE @i INT = 0;
WHILE @i <= 30
BEGIN
    DECLARE @name NVARCHAR(20) = 'nutc' + RIGHT('00' + CAST(@i AS VARCHAR), 2);
    DECLARE @sql NVARCHAR(200) = 'CREATE LOGIN [' + @name + '] WITH PASSWORD = N''Nutc@2026'', CHECK_POLICY = OFF';
    EXEC sp_executesql @sql;
    SET @i = @i + 1;
END
```
